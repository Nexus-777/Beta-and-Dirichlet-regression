# Beta-and-Dirichlet-regression
This repository contains the R code for my final project on **Beta regression** and **Dirichlet regression**.

The project looks at how these two methods can be used to model proportional data.  
In particular, it focuses on the difference between modeling a single proportion and modeling compositional outcomes with multiple parts.  
The code in this repository is used for model fitting, comparison, and presentation of results.

## Files

- **G7 model.Rmd**: main modeling work for the G7 analysis  
- **g7_multi_single_compare.Rmd**: comparison of multi-country and single-country models  
- **New table.Rmd**: code for producing summary tables and generate simple beta and Dirichlet models
- **README.md**: project description

## Project focus

The main focus of this project is:
- understanding the use of Beta regression for proportions in \((0,1)\);
- understanding the use of Dirichlet regression for compositional responses;
- comparing model fit under different model settings;
- presenting results clearly through tables and figures.

## Software

All analyses are written in **R**, mainly through **R Markdown** files.
